Specialization Choice (By Sep 25)

Recommended for you: AppSec / Bug Bounty. Why? You code. You understand how Node.js/Express handles input, queries, authentication. Most pentesting candidates can't read source code — you can. This is a superpower in interviews.

Alternative paths: SOC Analyst (most openings), Cloud Security (AWS focus), Malware Analysis (harder, impressive).

Weeks 1–2: Sep 25–Oct 8 (Project 1–2)

Project 1: Vulnerable Web App + Writeup
• Fork DVWA (Damn Vulnerable Web App) or build your own intentionally vulnerable Express app
• Document 5 vulnerabilities you've built in (SQLi, XSS, CSRF, weak auth, IDOR)
• Write detailed explanation: why each vuln exists, how it's exploited, how to fix it
• Post on GitHub with writeup (this is your portfolio piece)

Project 2: Python Network Scanner
• Build a port scanner that does: TCP connect scan, service detection (banners), OS fingerprinting
• Add: export results to JSON, colored CLI output, multi-threading
• Code structure: clean, commented, no copy-paste from Nmap
• Document: how port scanning works, why timing matters
• Time: 5 days of work

Project 3: Log Monitoring Dashboard
• Python backend that watches syslog/auth.log for suspicious patterns
• Flags: repeated failed logins, port scans, permission changes, unusual IPs
• Simple web frontend (React or even Flask template) to view alerts
• Deploy on a Raspberry Pi or AWS (bonus: cloud deployment experience)
• Time: 4 days of work

Project 4: Bug Bounty Submission
• Set up HackerOne / Bugcrowd accounts

• Time: ongoing, but target 1 submission by Oct 25
